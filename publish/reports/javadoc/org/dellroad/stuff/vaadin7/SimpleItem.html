Class SimpleItem<T>

  • java.lang.Object
    • org.dellroad.stuff.vaadin7.SimpleItem<T>

    All Implemented Interfaces:
    Item, Serializable, BackedItem<T>

    public class SimpleItem<T>
extends Object
implements BackedItem<T>
    Simple read-only Item implementation backed by a Java object.

    Item Propertys are defined by PropertyDefs and retrieved by a PropertyExtractor.

    Although the item properties are read-only from "above", they can be made mutable from "below" by modifying the backing object and invoking fireValueChange().

    See Also:
    AbstractQueryContainer, AbstractSimpleContainer, Serialized Form

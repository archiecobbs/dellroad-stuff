    public class EnumContainer<T extends Enum<T>>
extends SelfKeyedContainer<T>
    Container backed by the instances of an Enum type.

    Instances will auto-detect properties defined by @ProvidesProperty annotations (if any), and will also pre-define the following properties:

    • "name" - String property derived from Enum.name()
    • "value" - type T property that returns the Enum instance value itself
    • "ordinal" - Integer property derived from Enum.ordinal()
    • "toString" - String property derived from Enum.toString()
    See Also:
    EnumComboBox, Serialized Form

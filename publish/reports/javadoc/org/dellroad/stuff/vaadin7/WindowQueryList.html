Class WindowQueryList<T>

  • java.lang.Object
    • org.dellroad.stuff.vaadin7.WindowQueryList<T>
  • All Implemented Interfaces:
    QueryList<T>

    public class WindowQueryList<T>
extends Object
implements QueryList<T>
    QueryList implementation that only actually holds a portion, or "window" of a larger list. If a list element outside the "window" is accessed, an InvalidQueryListException is thrown, prompting another query.
    See Also:

Class GitRepository

  • java.lang.Object
    • org.dellroad.stuff.spring.AbstractBean
      • org.dellroad.stuff.pobj.distrib.GitRepository
  • All Implemented Interfaces:
    DisposableBean, InitializingBean

    public class GitRepository
extends AbstractBean
    Represents a git(1) repository and provides a few simple methods that wrap the command-line client and provide the functionality needed by Synchronizer. Only normal (i.e., non-bare) repositories are supported.

    This class assumes the repository already has all branches and remotes already configured, and that no other external process is accessing the git(1) repository without our knowledge.

    Instances are thread safe and guarantees exclusive access to the working directory to each GitRepository.Accessor while it is in use.

    Requires git version at least 1.7.4.

036/**
037 * Main class for Simple XML Persistence Objects (POBJ).
038 *
039 * <h3>Overview</h3>
040 *
041 * <p>
042 * A {@link PersistentObject} instance manages an in-memory database with ACID semantics and strict validation that is
043 * represented by a regular Java object graph (i.e., a root Java object and all of the other objects that it (indirectly)
044 * references). The object graph can be (de)serialized to/from XML, and this XML representation is used for persistence
045 * in the file system. The backing XML file is read in at initialization time and updated after each change.
046 *
047 * <p>
048 * All changes required copying the entire object graph, and are atomic and strictly serialized. In other words, the
049 * entire object graph is read and written by value. On the filesystem, the persistent XML file
050 * is updated by writing out a new, temporary copy and renaming the copy onto the original, using {@link File#renameTo
051 * File.renameTo()} so that changes are also atomic from a filesystem perspective, i.e., it's not possible to open an
052 * invalid or partial XML file (assuming filesystem renames are atomic, e.g., all UNIX variants). This class also
053 * supports information flowing in the other direction, where we pick up "out of band" updates to the XML file (see below).
054 *
055 * <p>
056 * This class is most appropriate for use with information that must be carefully controlled and validated,
057 * but that doesn't change frequently. Configuration information for an application stored in a {@code config.xml}
058 * file is a typical use case; beans whose behavior is determined by the configured information can subclass
059 * {@link AbstractConfiguredBean} and have their lifecycles managed automatically.
060 *
061 * <p>
062 * Because each change involves copying of the entire graph, an efficient graph copy operation is desirable.
063 * The {@linkplain AbstractDelegate#copy default method} is to serialize and then deserialize the object graph
064 * to/from XML in memory. See {@link org.dellroad.stuff.java.GraphCloneable} for a much more efficient approach.
065 *
066 * <h3>Validation</h3>
067 *
068 * <p>
069 * Validation is peformed in Java (not in XML) and defined by the provided {@link PersistentObjectDelegate}. This delegate
070 * guarantees that only a valid root Java object can be read or written. Setting an invalid Java root object via
071 * {@link #setRoot setRoot()} with throw an exception; reading an invalid XML file on disk will generate
072 * an error (or be ignored; see "Empty Starts" below).
073 *
074 * <h3>Update Details</h3>
075 *
076 * <p>
077 * When the object graph is updated, it must pass validation checks, and then the persistent XML file is updated and
078 * listener notifications are sent out. Listeners are always notified in a separate thread from the one that invoked
079 * {@link #setRoot setRoot()}. Support for delayed write-back of the persistent XML file is included: this
080 * allows modifications that occur in rapid succession to be consolidated into a single filesystem write operation.
081 *
082 * <p>
083 * Support for optimistic locking is included. There is a "current version" number which is incremented each
084 * time the object graph is updated; writes may optionally specify this number to ensure no intervening changes
085 * have occurred. If concurrent updates are expected, applications may choose to implement a 3-way merge algorithm
086 * of some kind to handle optimistic locking failures.
087 *
088 * <p>
089 * To implement a truly atomic read-modify-write operation without the possibility of locking failure, simply
090 * synchronize on this instance, e.g.:
091 *  <blockquote><pre>
092 *  synchronized (pobj) {
093 *      MyRoot root = pboj.getRoot();
094 *      root.setFoobar("new value");    // or whatever else we want to modify
095 *      pobj.setRoot(root);
096 *  }
097 *  </pre></blockquote>
098 *
099 * <p>
100 * Instances can also be configured to automatically preserve one or more backup copies of the persistent file on systems that
101 * support hard links (see {@link FileStreamRepository}). Set the {@link #getNumBackups numBackups} property to enable.
102 *
103 * <h3>"Out-of-band" Writes</h3>
104 *
105 * <p>
106 * When a non-zero {@linkplain #getCheckInterval check interval} is configured, instances support "out-of-band" writes
107 * directly to the XML persistent file by some other process. This can be handy in cases where the other process (perhaps hand
108 * edits) is updating the persistent file and you want to have the running Java process pick up the changes just as if
109 * {@link PersistentObject#setRoot setRoot()} had been invoked. In particular, instances will detect the appearance
110 * of a newly appearing persistent file after has starting without one (see "empty starts" below). In all cases, persistent
111 * objects must properly validate. To avoid reading a partial file the external process should write the file atomically by
112 * creating a temporary file and renaming it; however, this race window is small and in any case the problem is self-correcting
113 * because a partially written XML file will not validate, and so it will be ignored and retried after another
114 * {@linkplain #getCheckInterval check interval} milliseconds has passed.
115 *
116 * <p>
117 * A special case of this is effected when {@link PersistentObject#setRoot setRoot()} is never explicitly invoked
118 * by the application. Then some other process must be responsible for all database updates via XML file, and this class
119 * will automatically pick them up, validate them, and send out notifications to listeners.
120 *
121 * <h3>Empty Starts and Stops</h3>
122 *
123 * <p>
124 * An "empty start" occurs when an instance is {@linkplain #start started} but the persistent XML file is either missing,
125 * does not validate, or cannot be read for some other reason. In such cases, the instance will start with no object graph,
126 * and {@link #getRoot} will return null. This situation will correct itself as soon as the object graph is written via
127 * {@link #setRoot setRoot()} or the persistent file appears (effecting an "out-of-band" update). At that time, {@linkplain
128 * #addListener listeners} will be notified for the first time, with {@link PersistentObjectEvent#getOldRoot} returning null.
129 *
130 * <p>
131 * Whether empty starts are allowed is determined by the {@link #isAllowEmptyStart allowEmptyStart} property (default
132 * {@code false}). When empty starts are disallowed and the persistent XML file cannot be successfully read,
133 * then {@link #start} will instead throw an immediate {@link PersistentObjectException}.
134 *
135 * <p>
136 * Similarly, "empty stops" are allowed when the {@link #isAllowEmptyStop allowEmptyStop} property is set to {@code true}
137 * (by default it is {@code false}). An "empty stop" occurs when a null value is passed to {@link #setRoot setRoot()}
138 * (note, an invalid XML file appearing on disk does not cause an empty start).
139 * Subsequent invocations of {@link #getRoot} will return {@code null}. The persistent file is <b>not</b>
140 * modified when null is passed to {@link #setRoot}. When empty stops are disallowed, then invoking
141 * {@link #setRoot} with a {@code null} object will result in an {@link IllegalArgumentException}.
142 *
143 * <p>
144 * Allowing empty starts and/or stops essentially creates an "unconfigured" state represented by a null root object.
145 * When empty starts and empty stops are both disallowed, there is no "unconfigured" state: once started, {@link #getRoot}
146 * can be relied upon to always return a non-null, validated root object.
147 *
148 * <p>
149 * See {@link AbstractConfiguredBean} for a useful superclass that automatically handles starting and stopping
150 * based on the state of an associated {@link PersistentObject}.
151 *
152 * <h3>Shared Roots</h3>
153 *
154 * Each time {@link #setRoot setRoot()} or {@link #getRoot getRoot()} is invoked, a deep copy of the root object
155 * is made. This prevents external code from changing any node in the "official" object graph held by this instance,
156 * and allows invokers of {@link #getRoot getRoot()} to modify to the returned object graph without affecting other invokers.
157 * However, there may be cases where this deep copy is an expensive operation in terms of time or memory.
158 * The {@link #getSharedRoot getSharedRoot()} method can be used in these situations. This method returns the same
159 * root object each time it is invoked (this shared root is itself a deep copy of the "official" root). Therefore, only
160 * the very first invocation pays the price of a copy. However, all invokers of {@link #getSharedRoot getSharedRoot()}
161 * must treat the object graph as read-only to avoid each other seeing unexpected changes.
162 *
163 * <h3>Delegate Function</h3>
164 *
165 * <p>
166 * Instances must be configured with a {@link PersistentObjectDelegate} that knows how to validate the object graph
167 * and perform conversions to and from XML. See {@link PersistentObjectDelegate} and its implementations for details.
168 *
169 * <h3>Schema Changes</h3>
170 *
171 * <p>
172 * Like any database, the XML schema may evolve over time. The {@link PersistentObjectSchemaUpdater} class provides a simple
173 * way to apply and manage schema updates using XSLT transforms.
174 *
175 * <h3>Transaction Manager</h3>
176 *
177 * <p>
178 * If using Spring, consider using a {@link PersistentObjectTransactionManager} for transactional access to a
179 * {@link PersistentObject}.
180 *
181 * @param <T> type of the root object
182 * @see PersistentObjectDelegate
183 * @see PersistentObjectSchemaUpdater
184 * @see PersistentObjectTransactionManager
185 * @see AbstractConfiguredBean
186 */
187public class PersistentObject<T> {

464    /**
465     * Start this instance. Does nothing if already started.
466     *
467     * @throws PersistentObjectException if an error occurs
468     */
469    public synchronized void start() {
470
471        // Already started?
472        if (this.started)
473            return;
474
475        // Sanity check
476        if (this.streamRepository == null)
477            throw new PersistentObjectException("no file configured");
478        if (this.delegate == null)
479            throw new PersistentObjectException("no delegate configured");
480
481        // Create executor services
482        this.scheduledExecutor = Executors.newSingleThreadScheduledExecutor();
483        this.notifyExecutor = Executors.newSingleThreadExecutor();
484
485        // Read file (if it exists)
486        this.log.info(this + ": starting");
487        if (this.getFile().exists()) {
488
489            // Apply file
490            try {
491                this.applyFile(this.getFile().lastModified(), true);
492            } catch (PersistentObjectException e) {
493                if (!this.isAllowEmptyStart())
494                    throw e;
495                this.log.warn("empty start: unable to load persistent file `" + this.getFile() + "': " + e);
496            }
497        } else {
498
499            // Persistent file does not exist, so get default value from delegate
500            final T defaultValue = this.delegate.getDefaultValue();
501
502            // If no default value, we have an empty start situation, otherwise apply the default value
503            if (defaultValue == null) {
504                if (!this.isAllowEmptyStart()) {
505                    throw new PersistentObjectException("persistent file `" + this.getFile() + "' does not exist,"
506                      + " there is no default value, and empty starts are disallowed");
507                }
508                this.log.info(this + ": empty start: persistent file `" + this.getFile() + "' does not exist"
509                  + " and there is no default value");
510            } else {
511                this.log.info(this + ": persistent file `" + this.getFile() + "' does not exist, applying default value");
512                try {
513                    this.setRootInternal(defaultValue, 0, false, true, false);
514                } catch (PersistentObjectException e) {                             // e.g., validation failure
515                    if (!this.isAllowEmptyStart())
516                        throw e;
517                    this.log.warn("empty start: unable to apply default value: " + e);
518                }
519            }
520        }
521
522        // Start checking the file
523        if (this.checkInterval > 0) {
524            this.scheduledExecutor.scheduleWithFixedDelay(new Runnable() {
525                @Override
526                public void run() {
527                    PersistentObject.this.checkFileTimeout();
528                }
529            }, this.checkInterval, this.checkInterval, TimeUnit.MILLISECONDS);
530        }
531
532        // Done
533        this.started = true;
534    }

536    /**
537     * Stop this instance. Does nothing if already stopped.
538     *
539     * @throws PersistentObjectException if a delayed write back is pending and error occurs while performing the write
540     */
541    public synchronized void stop() {
542
543        // Already stopped?
544        if (!this.started)
545            return;
546
547        // Perform any lingering pending save now
548        if (this.cancelPendingWrite())
549            this.writeback();
550
551        // Stop executor services
552        this.log.info(this + ": shutting down");
553        this.scheduledExecutor.shutdown();
554        this.notifyExecutor.shutdown();
555        this.awaitTermination(this.scheduledExecutor, "scheduledExecutor");
556        this.awaitTermination(this.notifyExecutor, "notifyExecutor");
557
558        // Reset
559        this.scheduledExecutor = null;
560        this.notifyExecutor = null;
561        this.root = null;
562        this.sharedRoot = null;
563        this.writebackRoot = null;
564        this.timestamp = 0;
565        this.started = false;
566    }

580     *
581     * <p>
582     * In the situation of an empty start or empty stop, this instance is "unconfigured" and null will be returned.
583     * This can only happen if {@link #setAllowEmptyStart setAllowEmptyStart(true)} or {@link #setAllowEmptyStop

585     *
586     * <p>
587     * This returns a deep copy of the current root object; any subsequent modifications are not written back.
588     * Use {@link #getSharedRoot} instead to avoid the cost of the deep copy at the risk of seeing modifications
589     * caused by other invokers.
590     *

592     * @throws IllegalStateException if this instance is not started
593     * @throws PersistentObjectException if an error occurs
594     */

596
597        // Sanity check
598        if (!this.started)
599            throw new IllegalStateException("not started");
600
601        // Copy root
602        return this.root != null ? this.delegate.copy(this.root) : null;
603    }

638    /**
639     * Read the shared root object (as with {@link #getSharedRoot}) and its version (as with {@link #getVersion})
640     * in one atomic operation.
641     * This avoids the race condition inherent in trying to perform these operations separately.
642     *
643     * @return snapshot of the current shared root, or null during an empty start or empty stop
644     * @throws IllegalStateException if this instance is not started
645     * @throws PersistentObjectException if an error occurs
646     * @see #getSharedRoot
647     */

655    /**
656     * Atomically update the root object.
657     *
658     * <p>
659     * The given object is deep-copied, the copy replaces the current root, and the new version number is returned.
660     * If there is no write delay configured, the new version is written to the underlying file immediately and
661     * a successful return from this method indicates the new root has been persisted. Otherwise, the write will
662     * occur at a later time in a separate thread.
663     *
664     * <p>
665     * If {@code expectedVersion} is non-zero, then if the current version is not equal to it,
666     * a {@link PersistentObjectVersionException} exception is thrown. This mechanism
667     * can be used for optimistic locking.
668     *
669     * <p>
670     * If empty stops are allowed, then {@code newRoot} may be null, in which case it replaces the
671     * current root and subsequent calls to {@link #getRoot} will return null. When a null
672     * {@code newRoot} is set, the persistent file is <b>not</b> modified.
673     *
674     * <p>
675     * If the given root object is {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current
676     * root object, then no action is taken and the current (unchanged) version number is returned.
677     *
678     * <p>
679     * After a successful change, any registered {@linkplain PersistentObjectListener listeners} are notified in a
680     * separate thread from the one that invoked this method.
681     *
682     * @param newRoot new persistent object
683     * @param expectedVersion expected current version number, or zero to ignore the current version number
684     * @return the new current version number (unchanged if {@code newRoot} is
685     *  {@linkplain PersistentObjectDelegate#isSameGraph the same as} the current root)
686     * @throws IllegalArgumentException if {@code newRoot} is null and empty stops are disallowed
687     * @throws IllegalArgumentException if {@code expectedVersion} is negative
688     * @throws IllegalStateException if this instance is not started
689     * @throws PersistentObjectException if an error occurs
690     * @throws PersistentObjectVersionException if {@code expectedVersion} is non-zero and not equal to the current version
691     * @throws PersistentObjectValidationException if the new root has validation errors
692     */
693    public final synchronized long setRoot(T newRoot, long expectedVersion) {
694        return this.setRootInternal(newRoot, expectedVersion, false, false, false);
695    }
696
697    synchronized long setRootInternal(T newRoot, long expectedVersion,
698      boolean readingFile, boolean allowNotStarted, boolean alreadyValidated) {
699
700        // Sanity check
701        if (newRoot == null && !this.isAllowEmptyStop())
702            throw new IllegalArgumentException("newRoot is null but empty stops are not enabled");
703        if (!this.started && !allowNotStarted)
704            throw new IllegalStateException("not started");
705        if (expectedVersion < 0)
706            throw new IllegalStateException("negative expectedVersion");
707
708        // Check version number
709        if (expectedVersion != 0 && this.version != expectedVersion)
710            throw new PersistentObjectVersionException(this.version, expectedVersion);
711
712        // Check for sameness
713        if (this.root == null && newRoot == null)
714            return this.version;
715        if (this.root != null && newRoot != null && this.delegate.isSameGraph(this.root, newRoot))
716            return this.version;
717
718        // Validate the new root
719        if (newRoot != null && !alreadyValidated) {
720            Set<ConstraintViolation<T>> violations = this.delegate.validate(newRoot);
721            if (!violations.isEmpty())
722                throw new PersistentObjectValidationException(violations);
723        }
724
725        // Do the update
726        final T oldRoot = this.root;
727        this.root = newRoot != null ? this.delegate.copy(newRoot) : null;
728        this.version++;
729        this.sharedRoot = null;
730
731        // Perform write-back, either now or later
732        if (!readingFile && this.root != null) {
733            this.writebackRoot = this.root;
734            if (this.writeDelay == 0)
735                this.writeback();
736            else if (this.pendingWrite == null) {
737                this.pendingWrite = this.scheduledExecutor.schedule(new Runnable() {
738                    @Override
739                    public void run() {
740                        PersistentObject.this.writeTimeout();
741                    }
742                }, this.writeDelay, TimeUnit.MILLISECONDS);
743            }
744        }
745
746        // Notify listeners
747        this.notifyListeners(this.version, oldRoot, newRoot);
748
749        // Done
750        return this.version;
751    }

773    /**
774     * Check the persistent file for an "out-of-band" update.
775     *
776     * <p>
777     * If the persistent file has a newer timestamp than the timestamp of the most recently read
778     * or written version, then it will be read and applied to this instance.
779     *
780     * @throws IllegalStateException if this instance is not started
781     * @throws PersistentObjectException if an error occurs
782     */
783    public synchronized void checkFile() {
784
785        // Sanity check
786        if (!this.started)
787            throw new IllegalStateException("not started");
788
789        // Get file timestamp
790        long fileTime = this.getFile().lastModified();
791        if (fileTime == 0)
792            return;
793
794        // Check whether file has newly appeared or just been updated
795        if (this.timestamp != 0 && fileTime <= this.timestamp)
796            return;
797
798        // Read new file
799        this.log.info(this + ": detected out-of-band update of persistent file `" + this.getFile() + "'");
800        this.applyFile(fileTime, false);
801    }

871    /**
872     * Write the persistent file and rotate any backups.
873     *
874     * <p>
875     * A temporary file is created in the same directory and then renamed to provide for an atomic update
876     * (on supporting operating systems).
877     *
878     * @param obj root object to write
879     * @throws IllegalArgumentException if {@code obj} is null
880     * @throws PersistentObjectException if an error occurs
881     */
882    protected final synchronized void write(T obj) {
883
884        // Sanity check
885        if (obj == null)
886            throw new IllegalArgumentException("null obj");
887
888        // Open atomic update output and buffer it
889        AtomicUpdateFileOutputStream updateOutput;
890        try {
891            updateOutput = this.streamRepository.getOutputStream();
892        } catch (IOException e) {
893            throw new PersistentObjectException("error creating temporary file", e);
894        }
895        BufferedOutputStream output = new BufferedOutputStream(updateOutput);
896
897        // Serialize to XML
898        try {
899
900            // Set up XML result
901            Result result = this.createResult(output, updateOutput.getTempFile());
902
903            // Serialize root object
904            PersistentObject.write(obj, this.delegate, result);
905
906            // Commit output
907            try {
908                output.close();
909            } catch (IOException e) {
910                throw new PersistentObjectException("error closing temporary file", e);
911            }
912
913            // Success
914            output = null;
915        } finally {
916            if (output != null)
917                updateOutput.cancel();
918        }
919
920        // Update file timestamp
921        this.timestamp = this.streamRepository.getFileTimestamp();
922    }

940    /**
941     * Notify listeners of a change in value.
942     *
943     * @param newVersion the version number associated with the new root
944     * @param oldRoot previous root object
945     * @param newRoot new root object
946     */
947    protected void notifyListeners(long newVersion, T oldRoot, T newRoot) {
948
949        // Snapshot listeners
950        final ArrayList<PersistentObjectListener<T>> listenersCopy = new ArrayList<PersistentObjectListener<T>>();
951        synchronized (this.listeners) {
952            listenersCopy.addAll(this.listeners);
953        }
954
955        // Notify them
956        final PersistentObjectEvent<T> event = new PersistentObjectEvent<T>(this, newVersion, oldRoot, newRoot);
957        this.notifyExecutor.submit(new Runnable() {
958            @Override
959            public void run() {
960                PersistentObject.this.doNotifyListeners(listenersCopy, event);
961            }
962        });
963    }

965    // Read the persistent file and apply it
966    private synchronized void applyFile(long newTimestamp, boolean allowNotStarted) {
967        this.cancelPendingWrite();
968        this.timestamp = newTimestamp;                              // update timestamp even if update fails to avoid loops
969        this.setRootInternal(this.read(), 0, true, allowNotStarted, false);
970    }

972    // Handle a write-back timeout
973    private synchronized void writeTimeout() {
974
975        // Check for cancel race
976        if (this.pendingWrite == null)
977            return;
978        this.pendingWrite = null;
979
980        // Write it
981        try {
982            this.writeback();
983        } catch (ThreadDeath t) {
984            throw t;
985        } catch (Throwable t) {
986            this.delegate.handleWritebackException(this, t);
987        }
988    }

990    // Handle a check file timeout
991    private synchronized void checkFileTimeout() {
992
993        // Handle race condition
994        if (!this.started)
995            return;
996
997        // Check file
998        try {
999            this.checkFile();
1000        } catch (ThreadDeath t) {
1001            throw t;
1002        } catch (Throwable t) {
1003            this.log.error(this + ": error attempting to apply out-of-band update", t);
1004        }
1005    }

1007    // Write back root to persistent file
1008    private synchronized void writeback() {
1009        T objectToWrite = this.writebackRoot;
1010        assert objectToWrite != null;
1011        this.writebackRoot = null;
1012        this.write(objectToWrite);
1013    }

1015    // Cancel a pending write and return true if there was one
1016    private synchronized boolean cancelPendingWrite() {
1017        if (this.pendingWrite == null)
1018            return false;
1019        this.pendingWrite.cancel(false);
1020        this.pendingWrite = null;
1021        return true;
1022    }

1024    // Notify listeners. This is invoked in a separate thread.
1025    private void doNotifyListeners(ArrayList<PersistentObjectListener<T>> list, PersistentObjectEvent<T> event) {
1026        for (PersistentObjectListener<T> listener : list) {
1027            try {
1028                listener.handleEvent(event);
1029            } catch (ThreadDeath t) {
1030                throw t;
1031            } catch (Throwable t) {
1032                this.log.error(this + ": error notifying listener " + listener, t);
1033            }
1034        }
1035    }

1037    // Wait for an ExecutorService to completely shut down
1038    private void awaitTermination(ExecutorService executor, String name) {
1039        boolean shutdown = false;
1040        try {
1041            shutdown = executor.awaitTermination(EXECUTOR_SHUTDOWN_TIMEOUT, TimeUnit.MILLISECONDS);
1042        } catch (InterruptedException e) {
1043            this.log.warn(this + ": interrupted while awaiting " + name + " termination");
1044            Thread.currentThread().interrupt();
1045        }
1046        if (!shutdown)
1047            this.log.warn(this + ": failed to completely shut down " + name);
1048    }

1057     * to go through the whole {@link PersistentObject} lifecycle.
1058     *
1059     * @param <T> root object type
1060     * @param delegate delegate supplying required operations
1061     * @param source source for serialized root object
1062     * @param validate whether to also validate the root object
1063     * @return deserialized root object, never null
1064     * @throws IllegalArgumentException if any parameter is null
1065     * @throws PersistentObjectValidationException if {@code validate} is true and the deserialized root has validation errors
1066     * @throws PersistentObjectException if an error occurs
1067     */
1068    public static <T> T read(PersistentObjectDelegate<T> delegate, Source source, boolean validate) {
1069
1070        // Sanity check
1071        if (delegate == null)
1072            throw new IllegalArgumentException("null delegate");
1073        if (source == null)
1074            throw new IllegalArgumentException("null source");
1075
1076        // Parse XML
1077        T root;
1078        try {
1079            root = delegate.deserialize(source);
1080        } catch (IOException e) {
1081            throw new PersistentObjectException("error reading persistent object", e);
1082        }
1083
1084        // Check result
1085        if (root == null)
1086            throw new PersistentObjectException("null object returned by delegate.deserialize()");

1091            if (!violations.isEmpty())
1092                throw new PersistentObjectValidationException(violations);
1093        }
1094
1095        // Done

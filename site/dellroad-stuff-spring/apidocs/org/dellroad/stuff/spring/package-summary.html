Package org.dellroad.stuff.spring Description

Spring related classes.

This also package implements the <dellroad-stuff:sql> and <dellroad-stuff:sql-update> custom Spring XML elements; these define beans of type SQLCommandList and SpringSQLSchemaUpdate, respectively.

About DellRoad Stuff

DellRoad Stuff is a dumping ground for various useful Java classes that have evolved over time from repeated use in multiple projects.

Project Modules

This project has declared the following modules:

Name Description
DellRoad Stuff JiBX Classes DellRoad Stuff contains classes related to JiBX.
DellRoad Stuff Main DellRoad Stuff general purpose classes.
DellRoad Stuff Maven Plugin Maven plugin containing miscellaneous useful stuff.
DellRoad Stuff PersistentObject Classes DellRoad Stuff classes related to PersistenceObject, a very simple XML flat-file object database.
DellRoad Stuff Spring Classes DellRoad Stuff classes related to the Spring Framework.
DellRoad Stuff Test Classes DellRoad Stuff classes used for testing.
DellRoad Stuff Vaadin Parent Module DellRoad Stuff classes related to Vaadin (parent module).

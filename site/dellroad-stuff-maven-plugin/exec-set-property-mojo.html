dellroad-stuff:exec-set-property

Full name:

org.dellroad:dellroad-stuff-maven-plugin:2.5.4:exec-set-property

Description:

Sets a Maven project property based on the output of an arbitrary command.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: initialize.

Required Parameters

Name Type Since Description
<command> String - The command to execute.

Optional Parameters

Name Type Since Description
<directory> File - The current working directory for the command.
Default value is: ${project.basedir}.
User property is: directory.
<encoding> String - The name of the character encoding to use when converting the command's standard output bytes into a String.
Default value is: UTF-8.
User property is: encoding.
<parameters> List - The command's parameters, if any.
<propertyName> String - The name of the Maven property to set.
User property is: propertyName.
<trim> boolean - Whether to trim whitespace before setting the property value.
Default value is: true.
User property is: trim.

Parameter Details

<command>

The command to execute.
  • Type: java.lang.String
  • Required: Yes

<directory>

The current working directory for the command.
  • Type: java.io.File
  • Required: No
  • User Property: directory
  • Default: ${project.basedir}

<encoding>

The name of the character encoding to use when converting the command's standard output bytes into a String.
  • Type: java.lang.String
  • Required: No
  • User Property: encoding
  • Default: UTF-8

<parameters>

The command's parameters, if any.
  • Type: java.util.List
  • Required: No

<propertyName>

The name of the Maven property to set.
  • Type: java.lang.String
  • Required: No
  • User Property: propertyName

<trim>

Whether to trim whitespace before setting the property value.
  • Type: boolean
  • Required: No
  • User Property: trim
  • Default: true

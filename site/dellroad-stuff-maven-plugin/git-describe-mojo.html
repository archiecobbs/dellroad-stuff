dellroad-stuff:git-describe

Full name:

org.dellroad:dellroad-stuff-maven-plugin:2.4.10:git-describe

Description:

Sets a Maven project property based on the output of git describe.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: initialize.

        Iterable<? extends K> generateContainerObjects(T root)
        Generate the Java objects that will back the AbstractSimpleContainer associated with a PersistentObjectContainerLoader from the given PersistentObject root.

        This method should not modify root.

        If the associated PersistentObject allows empty starts or stops, then root might be null.

        Parameters:
        root - updated persistent object root, or null in the case of an empty start or stop
        Returns:
        container backing objects

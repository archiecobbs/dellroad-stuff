    public class FieldLayout<T>
extends HorizontalLayout
implements Field<T>
    A HorizontalLayout that also exposes an internally wrapped Field. This is useful when you have an existing Field and want to display it differently and/or with additional Components but still need it to function as a Field from a data perspective.

    The layout is initially empty; use HorizontalLayout.addComponent() to add content.

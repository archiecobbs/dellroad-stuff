    public abstract class VaadinApplicationListener<E extends ApplicationEvent>
extends VaadinExternalListener<ApplicationEventMulticaster>
implements ApplicationListener<E>
    A Spring ApplicationListener support superclass customized for use by listeners that are part of a Vaadin application when listening to non-Vaadin application event sources.

    Listeners that are part of a Vaadin application should use this superclass if they are going to be registered with non-Vaadin event sources. This will ensure that events are delivered in the proper Vaadin application context and memory leaks are avoided when the session closes.

    See Also:
    VaadinExternalListener, VaadinUtil.invoke(com.vaadin.server.VaadinSession, java.lang.Runnable), VaadinApplicationScope, SpringVaadinSessionListener

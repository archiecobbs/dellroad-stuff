    public class SimpleProperty<T,V>
extends ReadOnlyProperty<V>
implements BackedProperty<T,V>
    Simple read-only Property implementation backed by a Java object.

    The Property is defined using a PropertyDef along with a PropertyExtractor that is capable of reading the property's value from the underlying Java object.

    See Also:
    SimpleContainer, SimpleItem, Serialized Form
